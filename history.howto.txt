 1717  \paragraph{Predicting Transcription Factor Binding Using Deep Learning Methods.~~}
 1718  We have developed a deep learning-based method to predict the binding patterns of TFs across six different cell lines from the ENCODE project.  We developed a novel hybrid deep learning convolutional/parametric approach that combines information present in the DNA sequence with chromatin accessibility status at that same genomic coordinate.  The sequence model is based on a convolutional neural network consisting of learnable weights scanned along some input matrix. The chromatin shape model is based on the CENTIPEDE model with additional over-dispersion parameters that can also use the fragment length information in ATAC-seq data.  Our convolutional neural network model is composed of six layers and the output value represents the prior probability of a window being bound by a TF that is used in the CENTIPEDE model. Using this method, we trained a separate model for 20 TF  and cell type combinations with  ChIP-seq peaks.  In nearly every case, this deep learning model outperforms the CENTIPEDE model (AUC $>$0.9).
